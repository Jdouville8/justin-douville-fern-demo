# Getting Started

> Getting started with The Plant Room API is simple—and a lot of fun. Whether you’re building your first project or adding plant-powered features to an existing app, this guide will show you the quickest path from idea to real API calls. All you need is your preferred programming language, a few lines of code, and a curiosity for what’s in stock.

<Card className="intro-card">
  Below, you'll find examples in TypeScript, Python, Go, and Java to help you make your first request in seconds. Once you run your first call, you can explore plant data, experiment with searches, or start integrating real features into your application. The Plant Room API is designed to be friendly, flexible, and ready to grow with your imagination—so jump in and start building something delightful.

Here are some common langauge examples for use to call The Plant Room API and see what's in stock right now
<br></br>
  <Tabs>
    <Tab title="TypeScript">
```typescript
import { TheGreenRoomApiClient } from "@fern-api/plantstore";

const client = new TheGreenRoomApiClient();
const plants = await client.plant.searchPlantsByStatus({
  status: "available"
});
console.log(plants);
```
    </Tab>

    <Tab title="Python">
```python
from the_green_room import TheGreenRoomApi

client = TheGreenRoomApi()
plants = client.plant.search_plants_by_status(status="available")
print(plants)
```
    </Tab>

    <Tab title="Go">
```go
import (
    "context"
    thegreenroom "github.com/fern-api/plantstore-go"
)

client := thegreenroom.NewClient()
plants, err := client.Plant.SearchPlantsByStatus(
    context.TODO(),
    &thegreenroom.SearchPlantsByStatusRequest{
        Status: thegreenroom.String("available"),
    },
)
```
    </Tab>

    <Tab title="Java">
```java
import com.the_green_room.api.TheGreenRoomApiClient;
import com.the_green_room.api.resources.plant.types.PlantResponse;

TheGreenRoomApiClient client = TheGreenRoomApiClient.builder().build();
List<PlantResponse> plants = client.plant().searchPlantsByStatus("available");
System.out.println(plants);
```
    </Tab>
  </Tabs>

## API Endpoints
<br></br>

### Plant Operations
<br></br>
| Method | Description |
|--------|-------------|
| POST | Add a new plant to the store |
| PUT | Update an existing plant |
| GET | Search plants by status |
| GET | Search plants by tags |
| GET | Find plant by ID |
<br></br>

### User Operations
<br></br>
| Method | Description |
|--------|-------------|
| GET | Logs user into the system |
| GET | Logs out current logged-in user session |
| GET | Get user by username |
