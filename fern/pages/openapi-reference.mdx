# OpenAPI Reference

> The Plant Room API follows the OpenAPI Specification, a standard format for describing RESTful APIs that makes integration simple and predictable.

<Card className="intro-card">
  The OpenAPI Specification (formerly known as Swagger) provides a comprehensive framework for API documentation and tooling. It defines a standard, language-agnostic interface that allows both humans and computers to understand the capabilities of a service without needing access to source code or additional documentation.

  To learn more about the OpenAPI Specification and explore its full capabilities, visit the [official OpenAPI Specification documentation](https://swagger.io/specification/).
</Card>

  <Accordion title="Why OpenAPI?" defaultOpen={false}>
